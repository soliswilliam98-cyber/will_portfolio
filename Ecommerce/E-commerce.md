# E-commerce Sales Analysis Project (PostgreSQL)

## Objective
This project analyzes an e-commerce database to extract key business insights such as sales performance, customer behavior, and inventory status.

## Database Overview
The system includes the following tables:
- Customers
- Orders
- Order Items
- Products

## Tools Used
- PostgreSQL
- SQL

## Key Business Questions Answered
1. Top-selling products
2. Monthly revenue trends
3. Most profitable product categories
4. Average order value (AOV)
5. Top 5 customers by spending
6. Inactive customers (last 6 months)
7. Customer acquisition by country
8. Low stock products

## Key Insights
- Electronics products like laptops and smartphones are top sellers
- Revenue shows growth over time
- High-value customers contribute significantly to total sales
- Some customers are inactive and could be re-engaged
- Certain products are low in stock and need restocking

## Conclusion
This project demonstrates how SQL can be used to analyze real-world business data and generate actionable insights for decision-making.
