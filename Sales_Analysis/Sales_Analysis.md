# Sales Data Analysis Project (Python)

## Objective

The goal of this project is to analyze sales data and extract meaningful business insights using Python.

## Tools Used

* Python
* pandas
* matplotlib

## Data Cleaning

* Removed duplicates
* Converted data types (quantity, price, date)
* Handled missing values
* Standardized column names

## Feature Engineering

* Created a new column: Revenue = Quantity × Price

## Analysis Performed

1. Total Revenue
2. Average Revenue per Transaction
3. Top Revenue-Generating Products
4. Revenue by Category
5. Revenue by Region
6. Monthly Revenue Trends

## Key Insights

* Revenue remained stable across years (~$2.3M)
* A small number of products generate most revenue (Pareto principle)
* Some regions outperform others significantly
* Sales show seasonal patterns

## Business Recommendations

* Focus on high-performing products
* Align marketing with peak months
* Expand in low-performing regions
* Monitor trends for growth opportunities

## Conclusion

This project demonstrates how Python can be used to clean, analyze, and visualize real-world business data to support decision-making.
